Commandes IA Disponibles

Module de Traitement du Langage Naturel (TLN)

Analyse de Sentiment

Détecte le sentiment (positif, négatif, neutre) d'un bloc de texte donné.

/tln sentiment "Votre texte ici"
Arguments: texte (chaîne de caractères)

Résumé de Texte

Génère un résumé concis d'un document ou d'un paragraphe long.

/tln résumer "Votre document long..." [longueur_souhaitée]
Arguments: texte (chaîne de caractères), longueur_souhaitée (optionnel, nombre de phrases/mots)

Traduction

Traduit un texte d'une langue source vers une langue cible.

/tln traduire "Hello" en français
Arguments: texte (chaîne de caractères), langue_cible (chaîne de caractères), langue_source (optionnel)

Extraction d'Entités Nommées

Identifie et extrait les entités nommées (personnes, lieux, organisations) d'un texte.

/tln entités "Pierre Dupont travaille à Paris chez Google."
Arguments: texte (chaîne de caractères)

Module de Vision par Ordinateur (VO)

Description d'Image

Génère une description textuelle du contenu d'une image.

/vo décrire [url_image]
Arguments: url_image (chaîne de caractères) ou image téléchargée

Détection d'Objets

Identifie et localise les objets spécifiques dans une image ou une vidéo.

/vo détecter [url_image] "chat", "chien"
Arguments: url_image (chaîne de caractères), objets_à_détecter (liste, optionnel)

Reconnaissance Faciale

Détecte et identifie les visages présents dans une image.

/vo visage [url_image]
Arguments: url_image (chaîne de caractères)

Module de Génération de Contenu

Génération de Texte

Crée du texte cohérent et pertinent basé sur une invite donnée.

/generer texte "Écris une courte histoire sur un dragon et un chevalier."
Arguments: invite (chaîne de caractères), longueur_max (optionnel)

Génération d'Image

Crée une image à partir d'une description textuelle.

/generer image "Un chat astronaute jouant de la guitare sur la lune."
Arguments: description (chaîne de caractères), style (optionnel)

Module de Recommandation

Recommander Produit

Suggère des produits basés sur l'historique de l'utilisateur ou des critères spécifiques.

/reco produit [id_utilisateur] [catégorie]
Arguments: id_utilisateur (nombre), catégorie (chaîne de caractères, optionnel)

Recommander Article

Propose des articles de blog ou d'actualité pertinents.

/reco article [mots_clés]
Arguments: mots_clés (liste, optionnel)
